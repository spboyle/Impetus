# Impetus

Throw spears, not paper airplanes

## Why / what
Throwing spears sucks in vanilla, this copies the logic of BetterArchery / BowPlugin, but for spears so they fly faster and feel more practical as a weapon.

## Configuration
You can configure various spear throwing physics via [Offical BepInEx Configuration Manager](https://thunderstore.io/c/valheim/p/Azumatt/Official_BepInEx_ConfigurationManager/).

## Notes
The implementation looks for the spear throw attack animation when deciding to adjust the projectile physics. Therefore, custom weapons like [Javelins](https://thunderstore.io/c/valheim/p/blacks7ar/Javelins/) are also affected (IMO, yay!), but other custom weapons could be affected that you don't want to be. If you have better ideas on how to isolate this affect to spears, holla atcha boy.
